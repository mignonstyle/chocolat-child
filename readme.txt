Description
------------------------------------ */

Use these theme files to start the development of a child theme of 'Chocolat'.

/* -- How to use -- */

1. Download the chocolat-child.zip file to your local machine.

2. Select the Dashboard of WordPress > Appearance > themes, Add New.

3. Select the zip file that you downloaded use the Upload link in the top links row to upload. click on the Install Now.

4. Activate the theme.

Good luck!
